# Web 悬浮桌面宠物 🐱 v2.7.0

原生 HTML + CSS + JS 实现的轻量悬浮桌面宠物，支持挂件版与浏览器插件版。

## 两种形态

| 形态 | 用途 | 部署方式 |
|------|------|----------|
| **挂件版** | 嵌入任意网页 | 引入一行 `<script>` 即可 |
| **插件版** | 浏览器全局常驻 | 安装 Chrome 扩展 |

## 功能清单

### 🐱 宠物形象
- 6种emoji形象：🐱小猫 🐶小狗 🐰兔子 🐼熊猫 🦊狐狸 🐧企鹅
- 支持导入自定义图片作为宠物（PNG/JPG/GIF）
- 皮肤数据跨页面同步，刷新不丢失

### 🖱️ 交互系统
- 自由拖动，边缘吸附，物理回弹
- 移动超5px才算拖拽，否则算点击/双击
- 单击：随机语录 + 宠物动画
- 双击：打开宠物控制台
- 右键：快捷功能菜单

### 🎬 动画状态机
- 6种状态：idle / clicked / dragged / happy / walk / idle_action
- 优先级切换，闲置行为自动调度
- PNG序列帧 + GIF支持

### 💬 对话气泡
- 气泡显示在宠物头顶，带尖角指向
- 多类语录池：日常、互动、闲置、提醒
- 消息队列，不会重叠

### ⏰ 计时提醒
- 快速倒计时：5 / 10 / 15 / 30分钟、1 / 2小时（单击直接设置）
- 自定义分钟数
- 设闹钟：选择具体时间
- 提醒到点时宠物自动移到屏幕中间，晃动提示
- 一次性提醒触发后自动从列表移除
- 右上角浮动提醒列表卡片，显示进行中提醒数量和剩余时间

### 📝 快捷便签
- 添加、完成、置顶、删除
- 置顶便签常驻显示

### 🎮 迷你游戏（右键菜单）
- ✊✌️🖐️ 石头剪刀布：实时对战，记录胜负
- 🎲 掷骰子：摇骰动画，历史记录
- 🔢 猜数字：1-100猜数，提示大小
- 🃏 21点：经典扑克牌，要牌或停牌
- 🧠 记忆翻牌：12张牌配对，计步数
- 🔨 打地鼠：15秒限时，点击得分
- 🪙 抛硬币：正反面，翻转动画+历史
- 🔮 今日运势：运势指数 + 幸运提示
- 🃏 今日一卡：塔罗牌抽卡

### 🧠 情绪 & AI 行为
- 宠物有情绪系统：开心、兴奋、好奇、孤单、困倦、难过
- 情绪随交互、天气、时间自动变化
- **情绪视觉效果**：宠物外观随情绪实时变化
  - 开心：暖色调 + 轻微呼吸放大
  - 兴奋：高饱和 + 弹跳动画
  - 好奇：微微倾斜 + 提亮
  - 孤单：去饱和 + 轻微摇晃
  - 困倦：变暗 + 缓慢呼吸
  - 难过：灰暗 + 下垂动画
  - 头顶情绪指示器（emoji）
  - 背后情绪光环（渐变光晕）
  - 支持 emoji 形象和自定义导入图片
- 接入 AI 后，宠物能自主思考、主动说话
- 交互时 AI 根据情绪生成个性化反应
- 整点报时、天气变化时 AI 主动播报

### 🐾 宠物动作（右键菜单）
- 🌀 影分身：分裂出多个残影
- 💃 跳舞：左右摇摆弹跳
- 🔄 旋转：360度旋转
- 👋 招手：打招呼
- 💤 睡觉：ZZZ 气泡 + 呼吸动画
- 👀 偷看：缩小再弹出
- 🥱 伸懒腰：拉伸动画
- 💖 爱心：飘出爱心
- ✨ 闪光：环绕闪光特效
- 🫣 探头：左右探头探脑

### 💬 AI 聊天
- 接入 AI 大模型，让宠物能和你对话
- 内置 9 家厂商：OpenAI、DeepSeek、通义千问、智谱、Moonshot、百川、讯飞星火、Ollama、自定义
- 选择厂商自动填入 API 地址和模型列表，也可手动填写任意 OpenAI 兼容 API
- 右键菜单「和宠物聊天」打开对话窗口
- 对话历史保留，支持清空
- 宠物气泡同步显示 AI 回复

### 🌤️ 天气系统
- 自动IP定位城市，支持手动切换
- wttr.in 免费API，无需key
- 每小时自动更新
- 天气通知卡片：每次更新弹出，显示60秒后自动收起
- 天气特效渲染：

| 天气 | 宠物效果 |
|------|----------|
| ☀️ 晴天 | 金色太阳脉动光晕 |
| ⛅ 多云 | 漂浮云朵 |
| 🌧️ 下雨 | 乌云 + 雨滴下落 |
| 🌨️ 下雪 | 雪花飘落旋转 |
| ⛈️ 雷暴 | 乌云 + 闪电 + 暴雨 |
| 🌫️ 大雾 | 雾气漂浮 |

### 🎨 皮肤系统
- emoji皮肤一键切换
- 导入自定义图片（单张/多帧）
- 皮肤数据存储在 chrome.storage.local，跨页面同步
- 自定义皮肤支持删除

### ⚙️ 设置面板（双击宠物打开）
- 显示设置：缩放、透明度、边缘吸附、闲置频率
- 提醒设置：整点报时、静默时段、进行中提醒列表
- 皮肤管理：切换、导入、删除
- 数据管理：导入导出配置、重置数据

### 💾 数据持久化
- 所有配置存入 chrome.storage.local（插件版）/ localStorage（挂件版）
- 跨页面同步，刷新不丢失
- 支持一键导出/导入JSON备份

## 快速开始

### 挂件版
```html
<script src="pet-core.js"></script>
<script>
  const pet = new WebPet();
  pet.say('你好呀~');
</script>
```

### 插件版
1. 下载 `releases/web-pet-extension-v2.7.0.zip` 并解压
2. Chrome 打开 `chrome://extensions`
3. 开启「开发者模式」
4. 点击「加载已解出的扩展程序」→ 选择解压后的文件夹
5. 打开任意网页，右下角出现宠物 🐱

## 操作方式

| 操作 | 效果 |
|------|------|
| 左键点击 | 随机语录 + 宠物动画 |
| 左键双击 | 打开控制台（提醒/便签/皮肤） |
| 左键拖拽 | 自由移动位置 |
| 右键点击 | 快捷功能菜单 |
| 拖到边缘 | 自动吸附，靠近弹出 |
| 插件图标 | 弹出控制面板 |
